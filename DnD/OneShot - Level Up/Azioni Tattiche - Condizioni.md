# Azioni Tattiche — Riferimento Rapido
**Solo guerrieri = Fighter, Elfo, Nano**

## Movimento e Posizione

| Azione | Meccanica |
|---|---|
| **Carica** | Min 20' linea retta. +2 TxC, -2 CA fino al prossimo turno |
| **Ritirata difensiva** | Metà movimento. No attacco. No AoO dal nemico |
| **Difesa totale** | +2 CA per il round. No attacco. Metà movimento |
| **Attraversare alleati** | +5' (1□) per ogni alleato attraversato |
| **Passare posizione** | Chi arretra: metà mov, no attacco. Chi avanza: metà mov, sì attacco |
| **Prone (a terra)** | Nemici +2 TxC contro di te. Alzarsi = metà movimento |

## Manovre Speciali

| Azione | Meccanica |
|---|---|
| **Spinta** | Prova FOR contrapposta (1d20 + mod FOR). Vincitore sposta 5' |
| **Disarmo** | -4 TxC. Se colpisci: nemico lascia arma. No danni. Solo Guerrieri |
| **Parare (F)** | Rinunci a tutti gli attacchi. +4 CA per il round. Solo Guerrieri |
| **Colpo potente (F)** | -5 TxC. Se colpisci: +5 danni. Solo Guerrieri |

## Lotta

| Azione | Meccanica |
|---|---|
| **Afferrare** | TxC normale. Se colpisci: afferri. No danni. No altre azioni |
| **Liberarsi** | Prova FOR contrapposta. Se vinci: libero. Nemico ottiene AoO |

## Azioni Preparate

| Azione | Meccanica |
|---|---|
| **Azione pronta** | Dichiara trigger + azione. Se trigger scatta: agisci subito. Se no: azione persa |
| **Preparare per carica** | Dichiara prima di iniziativa. Se nemico carica: colpisci per primo, danno x2 (lance/aste) |

## Attacchi a Distanza

| Situazione | Modifica |
|---|---|
| **Distanza corta** | +1 TxC |
| **Distanza lunga** | -1 TxC |
| **Tiro oltre 1+ alleato** | -2 TxC. Tiro naturale 1 = colpisci alleato |
| **Copertura parziale** | -2 TxC (bersaglio +2 CA) |
| **Copertura quasi totale** | -4 TxC (bersaglio +4 CA) |
| **Copertura vs area (TS)** | Parziale +2 TS, Quasi totale +4 TS |

---

**Legenda:** (F) = Solo Guerrieri | © = Corridoio/spazi stretti | AoO = Attacco di Opportunità
